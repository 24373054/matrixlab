## üéØ Core Research Areas

### Blockchain Technology
Advanced research in consensus mechanisms, cross-chain transactions, smart contract security, and decentralized applications (DApps). We focus on improving blockchain scalability, security, and interoperability.

**Key Topics:**
- Consensus Algorithms (DPoS, PoS, BFT)
- Cross-chain Bridge Security
- Smart Contract Vulnerability Detection
- Blockchain Sharding Protocols
- Account De-anonymization

### Federated Learning
Pioneering multimodal federated learning frameworks for heterogeneous devices and cross-domain IoT applications. Our research enables privacy-preserving machine learning across distributed systems.

**Key Topics:**
- Multimodal Federated Learning
- Privacy-Preserving ML
- Heterogeneous Device Collaboration
- Edge-Cloud Federated Systems

### Web 3.0 Ecosystem
Building the next generation of decentralized web infrastructure with focus on incentive mechanisms, semantic detection, and evolutionary game theory applications.

**Key Topics:**
- Decentralized Applications (DApps)
- Token Economics & Incentive Design
- Web3 Security & Privacy
- Semantic Transaction Analysis

### Intelligent Edge Computing & IoT
Developing edge intelligence solutions for AIoT applications, enabling real-time processing and decision-making at the network edge.

**Key Topics:**
- Edge Intelligence
- AIoT Applications
- Distributed Computing
- Real-time Data Processing

### AI & Machine Learning
Applying artificial intelligence and large language models to blockchain security, smart contract analysis, and recommendation systems.

**Key Topics:**
- LLM for Smart Contract Security
- Recommendation Unlearning
- AI-driven Vulnerability Detection
- Multimodal AI Systems
